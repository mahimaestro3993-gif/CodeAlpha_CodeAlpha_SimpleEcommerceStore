# AlphaGram - Full-Stack Social Media Platform

A robust, full-stack social media web application built with the MERN stack. The platform features user authentication, real-time post creation, interactive feeds, and a secure cloud database architecture.

## 🚀 Features
* **User Authentication:** Secure signup and login APIs.
* **Content Feed:** Create, read, and manage user posts seamlessly.
* **Architecture:** Decoupled Frontend and Backend structures for clean scalability.
* **Security-First Configuration:** Environment variables utilize localized management (`.env`) to protect cloud resources.

## 🛠️ Tech Stack
* **Frontend:** React, HTML5, CSS3, JavaScript
* **Backend:** Node.js, Express.js
* **Database:** MongoDB Atlas (Cloud Tier)
